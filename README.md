Overview

## Usage

install.shを参考にライブラリを入れたり，コンシュマー，アクセストークンを手に入れる．

env.pyのTWEET_FILTER_WORDSに取得したいツイートが含む単語のリストを入れる．  
(e.g. TWEET_FILTER_WORDS=['あああ','いいい'])
